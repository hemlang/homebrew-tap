# Homebrew Tap for Hemlock

This is the official Homebrew tap for the [Hemlock](https://github.com/hemlang/hemlock) programming language and related tools.

## Installation

```bash
brew tap hemlang/homebrew-tap
brew install hemlock
brew install hpm
```

## Available Formulas

| Formula | Description |
|---------|-------------|
| `hemlock` | The Hemlock interpreter and compiler (hemlock, hemlockc) |
| `hpm` | Hemlock Package Manager |

## Updating

```bash
brew update
brew upgrade hemlock hpm
```

## Uninstalling

```bash
brew uninstall hemlock hpm
brew untap hemlang/tap
```
